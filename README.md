# 📖 About

StayFinder is a Software Engineering project developed as part of the AIUB Software Engineering course.

The platform is designed to connect travelers with property owners by providing a centralized solution for searching, booking, and managing accommodations. It supports multiple user roles including Customer, Host, and Super Admin while focusing on security, usability, and scalability.

> **Current Status:** Planning & Design Completed. Development will begin in the next phase.

# 🎯 Project Objectives

- Simplify property searching
- Secure online booking
- Easy property management
- User-friendly interface
- Centralized administration
- Scalable system architecture
